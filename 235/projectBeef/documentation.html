Project 3 Documentation

Proposal Stage

At the start of this game's development, I was inspired by a project created by Chase Call: https://people.rit.edu/ccc9775/235/project3/.

I thought that I could try and recreate this project, but with a really nice-looking dungeon that I made for Dungeon23 a couple of months back.

Prototype Stage

However, issues started appearing immediately as I learned that loading these assets into the game was going to be a lot more difficult than I first thought.

Once I fully wrapped my head around it though, I decided I would have to make huge changes to the final project.

Those changes were that I would focus on a Level Editor for the project and for the main map explorer, it would only have transitions between rooms with music playing when the player enters the dungeon.

Final Project Stage

There were certainly a lot of bumps in the road for this project, but I'm pretty happy with how it turned out.

I don't have all the rooms that I would like in it, but that is mostly because it is very annoying to add a tile into this game as you have to update like 4 different scripts or it won't work.

However, the main functionality is in, and I'm proud of it. Over the summer, I may update it in my free time.
